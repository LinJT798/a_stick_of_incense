# 游戏截图说明

请将游戏截图放在此目录下，命名如下：

- `home.png` - 首页场景截图（包含游戏标题和开始按钮）
- `intro.png` - 介绍场景截图（包含继续按钮）
- `gameplay.png` - 主游戏场景截图（显示玩家躲避箭矢的场景）
- `gameover.png` - 结算场景截图（显示坚持的香数）

## 截图建议

1. 分辨率：建议使用游戏原始分辨率 1512×982 或等比例缩放
2. 格式：PNG格式，保证画质清晰
3. 内容：确保截图能体现各场景的核心特色

## 如何截图

1. 在浏览器中运行游戏
2. 进入对应场景
3. 使用浏览器开发者工具或截图软件截取游戏画面
4. 保存为对应文件名
